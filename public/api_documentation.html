📘 وثائق API

الروابط التالية مخصّصة للتواصل بين الواجهة الأمامية (React) والخادم (Laravel).

🌐 قاعدة الرابط (Base URL)

http://127.0.0.1:8000/api

جميع الروابط التالية مسبوقة بهذه القاعدة.

1️⃣ تسجيل حساب جديد

POST /register

إنشاء حساب جديد. الدور (role) يتم تعيينه تلقائياً إلى user.

▶ نموذج الطلب (JSON)

{
  "name": "أحمد محمد",
  "email": "ahmed@example.com",
  "password": "123456"
}

◀ نموذج الاستجابة (نجاح - 201)

{
  "status": "success",
  "message": "تم إنشاء الحساب بنجاح",
  "data": {
    "user": {
      "id": 1,
      "name": "أحمد محمد",
      "email": "ahmed@example.com",
      "role": "user",
      "created_at": "2025-01-15T10:00:00.000000Z"
    },
    "token": "1|abcdef123456...",
    "token_type": "Bearer"
  }
}

2️⃣ تسجيل الدخول

POST /login

▶ نموذج الطلب (JSON)

{
  "email": "ahmed@example.com",
  "password": "123456"
}

◀ نموذج الاستجابة (نجاح - 200)

{
  "status": "success",
  "message": "تم تسجيل الدخول بنجاح",
  "data": {
    "user": { ... },
    "token": "2|xyz789...",
    "token_type": "Bearer"
  }
}

3️⃣ عرض أول 15 منحة نشطة

GET /top-scholarships

يعرض المنح التي لم تنتهِ (finished_date >= اليوم)، مرتبة من الأحدث، مع:
- اسم المنحة، التمويل، الدرجة، اسم المدينة، اسم التخصص
- start_status: نص يوضح "تبدأ في ..." أو "تبقت X يوم"
- photo_url: رابط أول صورة (أو null).

◀ نموذج الاستجابة (200)

{
  "status": "success",
  "message": "تم جلب أول 15 منحة بنجاح",
  "data": [
    {
      "id": 25,
      "scholarship_name": "منحة جامعة الملك سعود",
      "finance": "ممولة بالكامل",
      "degree": "ماجستير",
      "city_name": "الرياض",
      "specialization_name": "علوم الحاسب",
      "start_status": "تبدأ في 2025-09-01",
      "photo_url": "http://127.0.0.1:8000/storage/scholarships/photo1.jpg"
    },
    {
      "id": 24,
      "scholarship_name": "منحة حكومة الإمارات",
      "finance": "جزئية",
      "degree": "بكالوريوس",
      "city_name": "دبي",
      "specialization_name": "إدارة الأعمال",
      "start_status": "تبقت 23 يوم",
      "photo_url": null
    }
  ]
}
